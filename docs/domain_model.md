# Domain Model

## User

Represents the owner of the account.

---

## Primary Goal

The main career objective.

Examples:

* Backend Engineer
* AI Engineer

---

## Secondary Goal

Smaller objectives that support the primary goal.

Examples:

* Finish DSA
* Build 2 Projects

---

## Skill

A capability required to achieve a goal.

Examples:

* Java
* SQL
* DSA

---

## Learning Topic

A smaller unit of a skill.

Examples:

* Arrays
* Strings
* Trees

---

## Resource

A trusted learning material.

Examples:

* YouTube Playlist
* PDF
* Course
* Drive Link

---

## Job Application

Represents an application to a company.

May optionally belong to a goal.

---

## Interview Round

Represents a stage of a job application.

Examples:

* Online Assessment
* Technical Round
* HR Round
